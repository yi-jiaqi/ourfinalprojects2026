Guide

This game is a game where two blue squares chase you (an orange square) and you have to try to not get hit by them. The goal of the game is to be able to survive as long as you can. There is a timer on the top left of the screen that shows how many seconds that you have been alive. You can use the arrows to move the orange square around. After every thirty seconds you will enter boss mode where two more purple squares appear out of nowhere, they are faster then the blue squares and they will start to chase you. Boss mode will last for twenty seconds and then it will go away. The boss mode keeps repeating every thirty seconds. That is how you play this game. Making this game took a long time. At first it was just one blue square chasing you. Then I made it beter by adding the time in seconds so you can keep track of how long that you have been a live. Finally i added boss mode which I think helps make the game way more interesting. Anaway I hope that you like playing this game. Have fun!
